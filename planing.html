Estructura de HTML5

Contenido

En HTML existe una anotima o cuerpo que debemos conocer al momento de crear un sitio web. Lo primero que debemos conocer es el inicio de la etiqueta HTML dentro de esta etiqueta va las dos mas importantes quienes brindan la estructura principal del sitio el HEAD y BODY. En este contenido nos vamos a enfocar por la estructura basica en este caso lo que va dentro del body.
HEADER: El contenido de esta etiqueta corresponde a la cabecera, al titulo de nuestro sitio.
DOCTYPE html: Esta etiqueta indica que el documento esta bajo el estandar HTML5.
DIV: Esta etiqueta tambien divide contenido, con la diferencia que este se utiliza para aplicar clases y estilos.
BODY: Esta etiqueta delimita el cuerpo del documento y contiene todo aquello que podemos ver en el navegador.
HEAD: Esta etiqueta inidica el inicio de la cabecera del documento.
META: Esta etiqueta define el tipo de codificacion del documento.
TITLE: Esta etiqueta sirve para declarar el titulo del documento. Este titulo es el que aparece en el navegador.
LINK: Se utiliza para llamar o enlazar documentos javascript y css externos.
SECTION: Esta etiqueta engloba un seccion de texto, imagenes y otros elementos que tienen cierta relación entre ellos.
ARTICLE: Esta etiqueta sirve para ordenar o dividir el contenido, suele estar dentro de la etiqueta de SECTION.
MAIN: Esta etiqueta define el contenido principal del documento.
H1,H2,H3,H4,H5,H6: Estas etiquetas son elementos de la cabecera, en este caso seis niveles de cabecera.
