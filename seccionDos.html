Novedades de HTML5

Octubre 15, 2020 por Maria

Una de las novedades de HTML5 ha sido la posibilidad de tener mas de un H1 por documento.

Hasta la llegada de HTML5 el documento solo permitia un H1 por documento, pero con la nueva versión esto ha cambiado. La estructura basica de HTML5 se basa en el significado semántico, este permite el uso de H1 para cada una de las etiquetas semánticas básicas es decir section y article.
